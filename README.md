# Shoplane - Full-Stack E-Commerce Web Application

A modern e-commerce platform built with **Spring Boot**, **Java**, **MySQL**, **HTML/CSS/JS**, and **RESTful APIs**.

## Features
- User authentication and profile management.
- Browse products by categories: Clothing, Electronics, Accessories, Gaming, etc.
- Add to cart, checkout, and order management.
- Admin dashboard for CRUD operations on products and orders.
- Responsive design for desktop and mobile devices.
- API tested with **Postman**.

## Technologies
- **Backend:** Spring Boot, Java, RESTful APIs
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Testing:** Postman
